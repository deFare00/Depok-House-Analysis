# Depok House Market Analysis

A data analyst portfolio project focused on analyzing the housing market in Depok using SQL and Power BI. This project includes data cleaning, outlier handling, exploratory data analysis (EDA), business insight generation, and interactive dashboard visualization.

## Tools & Technologies

* SQL (MySQL)
* Power BI
* CSV Dataset
* Data Cleaning & Data Visualization

## Project Objectives

This project aims to analyze:

* The most expensive and affordable housing areas in Depok
* The relationship between house prices and property features
* Market segmentation based on house prices
* Investment insights using price per square meter analysis

## Business Questions

### Market Analysis

* Which area has the highest average house price?
* Which area has the lowest average house price?
* What is the average house price in Depok?

### Property Analysis

* Does larger building size always mean higher price?
* Does garage capacity affect house prices?
* Does the number of bedrooms influence house prices?

### Investment Insight

* Which locations provide the best value for money?
* Which areas have the best price per square meter?
* How is the Depok housing market segmented?

## Data Cleaning Process

Several outliers and unrealistic values were identified during the analysis process, including:

* Extremely large building size values
* Unrealistically low house prices

To improve analysis quality, filtering and business logic validation were applied before creating visualizations and insights.

## Dashboard Features

* KPI Cards
* Average Price by Location
* Scatter Plot: Building Size vs House Price
* House Market Segmentation
* Garage vs Average Price Analysis
* Bedroom vs Average Price Analysis
* Interactive Filters (Slicers)

## Key Insights

* Certain locations have significantly higher average house prices.
* Larger building size generally correlates with higher house prices.
* Houses with larger garage capacity tend to have higher average prices.
* The middle-market housing segment dominates the Depok property market.
* Several areas offer better value based on price per square meter analysis.
